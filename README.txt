==== HOW TO RUN THE PROGRAM ====
In order to run the program, please cd into the directory of the application. From there, enter:
    
    javac Interfacer.java 

To compile the Interfacer class. 
From there, you can run this command:

    java Interfacer 

That will run the Interfacer class and prompt you with a welcome screen with several options displayed. Read through the options and enter the command of your choosing.
